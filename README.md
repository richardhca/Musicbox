# MusicBox

CMPT 470 D1 Project 

How to run:

1- Go to project root

2- Run `npm install`

3- Run `node ./bin/www`

4- Go to http://localhost:3000



##### NOTE: For now, you need postgres installed and running locally. The app expects postgres to run on port 5432 (i think it's the default)
